# 3.15更新
1. 清除代码中Token部分

2. 将非通用字段时间格式改为字符串，通过时间戳存储及传输

3. 增加订单完成时间更新接口（更新订单同时更新该订单下所有产品的数据）

4. 增加订单删除接口（删除订单同时删除该订单下所有产品的数据）

5. 创建数据表，记录商品打包映射关系

   | 段名       | 类型   | 含义                 |
   | ---------- | ------ | -------------------- |
   | id         | string | 主键，自动生成       |
   | big_size   | string | 相对大包装           |
   | small_size | string | 相对小包装或单个商品 |

   

6. 创建包装盒数据表

   | 段名 | 类型   | 含义           |
   | ---- | ------ | -------------- |
   | id   | string | 主键，自动生成 |
   | size | int    | 尺寸           |

7. 增加包装盒批量生成接口

8. 增加记录包装盒库存统计数据表

   | 段名   | 类型   | 含义     |
   | ------ | ------ | -------- |
   | id     | string | 主键     |
   | size   | int    | 容量     |
   | number | int    | 剩余数量 |

   

9. 批量创建包装盒同时应更新库存统计表
